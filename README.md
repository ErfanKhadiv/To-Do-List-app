# To-Do List Project

A simple and clean **To-Do List Web Application** built using **HTML**, **CSS**, and **JavaScript**. This project allows users to create tasks, set start and end times, mark tasks as complete, and clear tasks.

## ğŸ“„ Files Description

### **index.html**
Contains the main structure of the web app, including:
- Left panel with profile, clock, task input, and footer
- Right panel with task lists and buttons
- Icon support using FontAwesome
- Google Fonts integration

### **style.css**
Defines all styling, such as:
- Page layout (left/right sections)
- Buttons and input fields
- Task items and animations
- Footer and navigation styles

### **clock.js**
Updates the live digital clock every second using JavaScript.

### **index.js**
Handles the app functionality:
- Add, remove, and complete tasks
- Collapse sections (Tasks / Completed Tasks)
- Complete All and Clear All actions
